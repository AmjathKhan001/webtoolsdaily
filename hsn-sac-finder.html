What are HSN and SAC Codes?

The **Harmonized System of Nomenclature (HSN)** is an internationally recognized classification system for goods. It is a six-digit code that classifies more than 5,000 commodities, and the extended Indian system uses **eight digits**. It helps in systematic classification for taxation (GST) and customs purposes.

The **Service Accounting Code (SAC)** is a classification system used in India for services under the GST regime. SAC codes are always **six digits** and help in consistent tax application across various types of services, such as construction, consulting, and IT services.

Using the correct code is essential for accurate GST invoice generation and compliance.
